get /deployments

Get a list of deployments, i.e. candidates who have or are expected to enter into a deployment with this employer (i.e. list of hires)

  • Request
  • Response

HTTP status code 200

Body

Type: application/json

Schema:

{
	"$schema":"http://json-schema.org/draft-04/schema#",
	"type":"object",
	"additionalProperties":false,
	"properties":{
		"kind":{
			"enum":[
				"deployments"
			],
			"description":"always set to: deployments"
		},
		"items":{
			"type":"array",
			"items":{
				"oneOf":[
					{
						"$ref":"deployment.json"
					}
				]
			}
			
		}
		
	},
	"required":[
		"kind",
		"items"
	]
}

Example:

{
	"kind":"deployments",
	"items":[
		{
			"kind":"deployment",
			"id":102234,
			"status":"CONFIRMED",
			"recruitmentActivity":10234,
			"dutyEntryDate":"2002-10-02T10:00:00-05:00"
		}
	]
}
