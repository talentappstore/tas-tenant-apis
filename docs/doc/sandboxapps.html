About the sandbox apps

The sandbox apps allow you to develop your app without needing access to a full ATS.

Because the sandbox apps are private, you need to install them from an install token. This page lists the sandbox apps with their tokens. You don't normally need to install all of the apps below - each recipe will list the actual apps you need.

To install the sandbox apps you require, start by going to www.talentappstore.com, sign up and click at top right to create a tenant (use google auth to avoid a temporary issue with email-based logins).
