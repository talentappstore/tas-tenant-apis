/candidates/{candidateId}/resumes get

get

Get a list of all candidate resumes. Only candidates with a resume appear in the list.

get /candidates/{candidateId}/resumes

Get a list of all candidate resumes. Only candidates with a resume appear in the list.

  • Request
  • Response

URI Parameters

  • candidateId: required (string)

HTTP status code 200

Body

Type: application/json

Schema:

{  
          "type":"object",
          "$schema":"http://json-schema.org/draft-04/schema",
          "id":"http://jsonschema.net",
          "properties":{  
            "resume":{  
              "type":"object",
              "properties":{  
                "file":{  
                  "type":"string"
                },
                "content":{  
                  "type":"string"
                }
              },
              "required":[  
                "file",
                "content"
              ]
            }
          },
          "required":[  
            "resume"
          ]
        }

Example:

{  
          "resume":{  
            "candidateId":200,
            "file":{  
              "content":"juty876jhglk8976/0+jiyui756",
              "name":"resumeForBarry.doc"
            }
          }
        }
