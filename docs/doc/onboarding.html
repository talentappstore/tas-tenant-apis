Overview

This document is intended to help developers present their HR SaaS product as an onboarding app in Talent App Store.

Billing and financials

Billing for use of the onboarding API will be negotiated between the ATS vendor (producing new hires) and onboarding vendor.

As for any app in Talent App Store, your app also pays for core API calls, as described at http://developer.talentappstore.com. These costs are low.

See the onboarding API in action

Lets install a test app to demonstrate how the onboarding API works (if you're creating a new tenant, you may need to click the first install link twice).

Go to www.talentappstore.com, sign up and create a tenant (use google auth to avoid a temporary issue with email-based logins).

Install the Bitza app.

Click open on the Bitza app. YOu should now be able to test your onboarding app by following the instructions on the screen.
